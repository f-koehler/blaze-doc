Detailed Description

template<typename MT, bool SO>
class blaze::DMatDeclLowExpr< MT, SO >

Expression object for the explicit lower declaration of dense matrices.

The DMatDeclLowExpr class represents the compile time expression for the explicit lower declaration of a dense matrix.
