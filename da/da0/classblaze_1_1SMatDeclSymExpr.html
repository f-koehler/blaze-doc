Detailed Description

template<typename MT, bool SO>
class blaze::SMatDeclSymExpr< MT, SO >

Expression object for the explicit symmetry declaration of sparse matrices.

The SMatDeclSymExpr class represents the compile time expression for the explicit symmetry declaration of a sparse matrix.
