Detailed Description

template<typename MT1, typename MT2>
class blaze::TDMatTDMatMultExpr< MT1, MT2 >

Expression object for transpose dense matrix-transpose dense matrix multiplications.

The TDMatTDMatMultExpr class represents the compile time expression for multiplications between two column-major dense matrices.
