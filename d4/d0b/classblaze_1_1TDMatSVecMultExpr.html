Detailed Description

template<typename MT, typename VT>
class blaze::TDMatSVecMultExpr< MT, VT >

Expression object for transpose dense matrix-sparse vector multiplications.

The TDMatSVecMultExpr class represents the compile time expression for multiplications between column-major dense matrices and sparse vectors.
