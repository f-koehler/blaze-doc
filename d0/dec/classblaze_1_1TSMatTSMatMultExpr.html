Detailed Description

template<typename MT1, typename MT2>
class blaze::TSMatTSMatMultExpr< MT1, MT2 >

Expression object for transpose sparse matrix-transpose sparse matrix multiplications.

The TSMatTSMatMultExpr class represents the compile time expression for multiplications between two column-major sparse matrices.
