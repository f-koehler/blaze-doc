Detailed Description

template<typename VT, typename OP, bool TF>
class blaze::DVecForEachExpr< VT, OP, TF >

Expression object for the dense vector forEach() function.

The DVecForEachExpr class represents the compile time expression for the evaluation of a custom operation on each element of a dense vector via the forEach() function.

Compilation switch for the serial evaluation strategy of the for-each expression.

The useAssign compile time constant expression represents a compilation switch for the serial evaluation strategy of the for-each expression. In case the given dense vector expression of type VT requires an intermediate evaluation, useAssign will be set to 1 and the for-each expression will be evaluated via the assign function family. Otherwise useAssign will be set to 0 and the expression will be evaluated via the subscript operator.
